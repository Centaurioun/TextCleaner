PLIST Tag: Key

Wraps the selected text in a given tag.

SampleKey:

<key>SampleKey</key>

PLIST Tag: String

Wraps the selected text in a given tag.

Sample Text:

<string>Sample Text</string>

PLIST Tag: Integer

Wraps the selected text in a given tag.

1234:

<integer>1234</integer>

PLIST Tag: Float

Wraps the selected text in a given tag.

12.34:

<float>12.34</float>

PLIST Tag: True

Inserts the true XML tag.

<true/>
