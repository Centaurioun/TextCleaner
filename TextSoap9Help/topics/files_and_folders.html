Files and Folders

TextSoap places files in various locations to utilize certain features of macOS. The Application Support folder contains a number of files used by TextSoap, including a database that contains MyLibrary.

-

TextSoap Application Support folder:

~/Library/Application Support/TextSoap/

Preference file used by TextSoap:

~/Library/Preferences/com.unmarked.textsoap9.plist

For Setapp:

~/Library/Preferences/com.unmarked.textsoap-setapp.plist
