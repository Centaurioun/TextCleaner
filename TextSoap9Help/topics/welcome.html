System Requirements

TextSoap 9 built for macOS 13 (Ventura). Supports macOS 11 or later.

Welcome

TextSoap is a automated text processing tool that removes the tediousness of tasks associated with ‘cleaning up’ various text including removing extraneous characters, wrapping broken paragraphs, fixing quotation marks, finding and replacing patterned text, and more. It comes with more than 100 pre-built solutions and lets you create your own.

Terminology

Custom Cleaner: - user-defined series of actions that can apply a pre-built cleaner or use one of the many building blocks like regular expression find and replace, changing style, etc.

Custom Group - user-defined collection of “cleaners”. The collection can contain pre-built or custom cleaners.

See glossary for additional terms.

Using TextSoap

TextSoap is split into two components:

TextSoap

The main app that allows you to interactively process clipboard contents, text documents, create and edit custom cleaners and groups. This app comes in various editions: Direct and Setapp.

textsoapAgent

A free companion app that works with TextSoap to integrate features throughout your Mac. There is one version of and it interacts with whichever edition you are using.

  • TextSoap Menu : universal popup provides access to all of TextSoap cleaners to most apps
  • AppleScripting : script interface to provide cleaning via AppleScript
  • macOS Services : Optional service item brings up a dialog to use any cleaner within apps that support services
  • Custom Services : Using a group, select a small collection of cleaners to access via the services contextual menu
  • Hot Key Action: Using a global hot key, will copy/clean/paste the text into most apps

The agent is a separate download, which you find at https://textsoap.com/agent/
